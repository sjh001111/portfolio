# Joonghyuk Seong Portfolio

Personal portfolio site for Joonghyuk Seong. The site highlights current public work, archived projects, technical background, and downloadable resume materials.

## Current direction

- Built around a simple split between active public work and archived projects
- `JobMatch` has been moved into the archive and its public demo has been retired
- GitHub-facing repo copy now reflects the portfolio instead of the default Next.js starter text

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Vercel

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000` to view the site locally.

## Deploy

Production site: `https://joonghyukseong.com`
